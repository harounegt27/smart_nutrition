# Diet Recommendation System — End-to-End Machine Learning Application

Python • React • FastAPI • Docker • MLflow

University: Tek-Up — Guided ML Project (Python for Data Science 2)

Author: Mohamed Haroun Mezned

## Project Overview

This project is a full End-to-End Machine Learning System that performs:

1- Prediction of daily recommended macronutrients (Regression model)

2- Generation of a personalized meal plan (Generated by an external LLM based on predicted macronutrients)

The system integrates:

• Data preparation & automated feature engineering

• Full ML Pipeline (SMOTE, Preprocessing, Hyperparameter Tuning)

• MLflow experiment tracking

• FastAPI backend for predictions

• Frontend for interactive UI

• Docker containerization for deployment

## Dataset
Dataset used: Personalized Medical Diet Recommendations(from Kaggle)

Link → https://www.kaggle.com/datasets/ziya07/personalized-medical-diet-recommendations-dataset

The dataset contains patients’ medical profile, lifestyle indicators, nutritional intake, allergies, and the recommended diet plan.

| Column                   | Type        | Description                     |
| ------------------------ | ----------- | ------------------------------- |
| Age                      | Numeric     | Patient age                     |

## LLM-Based Meal Plan Generator

The system uses an external LLM API to generate:

✔Breakfast

✔ Lunch

✔ Dinner

✔ Snacks

✔ Macro totals matching the predicted ML values

✔ Optional explanation

The LLM receives only numeric targets predicted by the regression model:

Example prompt sent to LLM:

    {
      "calories": 2000,
      "protein": 130,
      "carbs": 210,
      "fat": 70
    }

.env Configuration :

    LLM_API_KEY=your_api_key_here
    LLM_PROVIDER=openai   # or gemini, groq, etc.
    MODEL_NAME=llama-3.3-70b-versatile
  
## Backend (FastAPI)
### Endpoints
| Method | Route             | Description                        |
| ------ | ----------------- | ---------------------------------- |
| GET    | `/`               | Health check                       |
| POST   | `/predict/macros` | Predict recommended macronutrients |
| POST   | `/diet/llm`       | Generate meal plan using LLM       |
| POST   | `/predict/batch`  | Batch inference                    |

### Run FastAPI
cmd :
    
    uvicorn app:app --reload --host 0.0.0.0 --port 8000

Access Swagger Docs → http://localhost:8000/docs

## Installation & Setup
1.Clone Projet:

    git clone https://github.com/harounegt27/diet_recommendation_system
    cd diet_recommendation_system

2.Backend Setup:

    python -m venv venv
    source venv/bin/activate       # Windows: venv\Scripts\activate
    pip install -r requirements.txt

3.Frontend Setup:

    cd frontend
    npm install

4.Run in Dev Mode:

  • Backend :

    cd frontend
    ng serve

• Frontend :
  
    cd code
    uvicorn app:app --reload
